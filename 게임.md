지금까지 배운 리지드바디, 콜라이더, 레이캐스트의 모든 기초 코드와 콤마(,)별 매개변수 명세서, 그리고 이 세 가지를 현실적으로 조합한 종합 예제 스크립트까지 한 장으로 깔끔하게 정리해 드릴게요.

1. 한눈에 보는 함수별 콤마(,) 명세서
① 힘 주기 (AddForce)
```C#
rb.AddForce(Vector3.up * 5f, ForceMode.Impulse);
//          [  값 1  ] [  값 2  ]
값 1 (Vector3.up * 5f): 힘의 방향과 크기 (어디로 얼마나 세게 칠 것인가)

값 2 (ForceMode.Impulse): 힘의 종류 (순간적인 충격량 Impulse, 지속적인 힘 Force 등)
```
② 충돌 감지 (OnCollisionEnter / OnTriggerEnter)
```C#
private void OnCollisionEnter(Collision collision) // 일반 물리 충돌 (벽, 바닥)
private void OnTriggerEnter(Collider other)       // 트리거 통과 충돌 (아이템, 포탈)
```
괄호 안 (collision / other): 부딪힌 상대방의 정보 상자. 유니티가 충돌 순간에 상대의 게임 오브젝트, 태그, 컴포넌트 정보 등을 이 상자에 담아서 던져줍니다.

③ 레이저 쏘기 (Physics.Raycast)
```C#
Physics.Raycast(transform.position, transform.forward, out hit, 10f);
//              [     값 1     ]  [     값 2    ] [ 값 3 ] [값 4]
값 1 (transform.position): 시작 위치 (어디서 쏠 것인가)

값 2 (transform.forward): 발사 방향 (어느 방향으로 쏠 것인가)

값 3 (out hit): 결과 저장 상자 (out 키워드를 붙여서, 레이저에 맞은 대상의 상세 정보를 hit이라는 변수에 자동으로 채우도록 함)

값 4 (10f): 최대 사거리 (얼마나 멀리 쏠 것인가, 미터 단위)
```
2. 삼총사를 하나로 합친 종합 예제 스크립트
이 스크립트는 "바닥을 체크해서 점프하고(Raycast + Rigidbody), 적을 밟았을 때 해치우거나 아이템을 먹는(Collider)" 실제 게임에서 가장 많이 쓰는 구조입니다.
```C#
using UnityEngine;

public class PlayerController : MonoBehaviour
{
    private Rigidbody rb;
    
    [Header("이동 관련 설정")]
    public float moveSpeed = 5f;
    public float jumpForce = 8f;

    [Header("레이캐스트 설정")]
    public float groundCheckDistance = 1.1f; // 캐릭터 중심에서 바닥까지의 거리

    void Start()
    {
        // [리지드바디] 내 오브젝트에 붙어있는 컴포넌트 가져오기
        rb = GetComponent<Rigidbody>();
    }

    void Update()
    {
        // 1. 점프 로직 (리지드바디 + 레이캐스트 조합)
        if (Input.GetKeyDown(KeyCode.Space))
        {
            // 내 위치(transform.position)에서 아래 방향(Vector3.down)으로 레이저를 쏩니다.
            // 바닥에 부딪혔는지 여부만 확인할 때는 out hit를 생략할 수도 있습니다.
            bool isGrounded = Physics.Raycast(transform.position, Vector3.down, groundCheckDistance);

            // 개발자 화면에 레이저를 그려서 시각적으로 확인 (선택 사항)
            Debug.DrawRay(transform.position, Vector3.down * groundCheckDistance, Color.red);

            if (isGrounded)
            {
                // [리지드바디] 바닥에 서 있을 때만 위쪽으로 순간적인 힘을 줍니다.
                rb.AddForce(Vector3.up * jumpForce, ForceMode.Impulse);
                Debug.Log("점프 성공!");
            }
        }
    }

    void FixedUpdate()
    {
        // 2. 이동 로직 (리지드바디 속도 제어)
        float xInput = Input.GetAxis("Horizontal");
        
        // 좌우는 입력값대로 속도를 고정하고, 상하(y)는 중력 엔진이 계산한 기존 속도를 그대로 유지합니다.
        rb.velocity = new Vector3(xInput * moveSpeed, rb.velocity.y, 0);
    }

    // 3. [콜라이더 - 일반 충돌] 물리적으로 부딪혔을 때 처리
    private void OnCollisionEnter(Collision collision)
    {
        // 부딪힌 상대방(collision)의 태그가 "Enemy" 라면
        if (collision.gameObject.CompareTag("Enemy"))
        {
            Debug.Log(collision.gameObject.name + "번 적과 부딪혀 데미지를 입었습니다!");
        }
    }

    // 4. [콜라이더 - 트리거 충돌] 겹치거나 통과했을 때 처리 (Is Trigger 체크된 오브젝트)
    private void OnTriggerEnter(Collider other)
    {
        // 나를 통과해간 상대방(other)의 태그가 "Item" 이라면
        if (other.CompareTag("Item"))
        {
            Debug.Log("아이템 획득: " + other.gameObject.name);
            Destroy(other.gameObject); // 상대 아이템 오브젝트를 파괴합니다.
        }
    }
}```
유니티 물리 시스템의 핵심 삼총사인 리지드바디(Rigidbody), 콜라이더(Collider), 레이캐스트(Raycast)의 기초 코드와 활용법을 정리해 드릴게요.

이 세 가지가 어디에 어떻게 쓰이는지 한눈에 이해할 수 있도록 구조를 먼저 잡고 코드로 들어갑시다!

리지드바디 (Rigidbody): "나 몸무게가 있고 중력을 받아!" (물리적인 몸을 부여)

콜라이더 (Collider): "나한테 부딪히면 아파!" (충돌할 수 있는 표면/부피를 부여)

레이캐스트 (Raycast): "내 눈에서 레이저 빔을 쏜다!" (눈에 보이지 않는 선을 쏴서 앞에 뭐가 있는지 감지)

1. 리지드바디 (Rigidbody)
오브젝트에 물리(중력, 힘)를 적용하고 제어할 때 사용합니다.

📌 어디에 배치하나요?
움직이는 오브젝트(플레이어, 적, 굴러가는 공, 총알 등)에 컴포넌트로 추가합니다.

💻 기초 코드 예시 (이동과 점프)
C#
using UnityEngine;

public class PlayerPhysics : MonoBehaviour
{
    private Rigidbody rb;
    public float moveSpeed = 5f;
    public float jumpForce = 7f;

    void Start()
    {
        // 1. 내 오브젝트에 붙은 Rigidbody 컴포넌트를 가져와 연결합니다.
        rb = GetComponent<Rigidbody>();
    }

    void FixedUpdate() // 물리 연산은 FixedUpdate에서 하는 것이 안전합니다.
    {
        // 2. 키보드 입력에 따라 좌우로 속도(Velocity)를 직접 줍니다.
        float xInput = Input.GetAxis("Horizontal");
        rb.velocity = new Vector3(xInput * moveSpeed, rb.velocity.y, 0);
    }

    void Update()
    {
        // 3. 스페이스바를 누르면 위쪽으로 팍! 하고 힘(AddForce)을 줍니다.
        if (Input.GetKeyDown(KeyCode.Space))
        {
            rb.AddForce(Vector3.up * jumpForce, ForceMode.Impulse);
        }
    }
}
2. 콜라이더 (Collider)
충돌을 감지하는 컴포넌트입니다. 코드로 작성할 때는 "충돌이 일어난 순간 자동으로 실행되는 메서드"를 사용합니다.

콜라이더는 크게 두 가지 모드가 있습니다.

일반 충돌 (딱딱한 벽): 서로 통과 못 하고 튕겨 나감 (OnCollision 계열 사용)

트리거 충돌 (통과하는 영역): 뚫고 지나가지만 감지는 됨 (OnTrigger 계열 사용, Collider 인스펙터창에서 Is Trigger 체크 필수)

📌 어디에 배치하나요?
충돌을 해야 하는 모든 오브젝트(바닥, 벽, 플레이어, 아이템, 함정 등)에 추가합니다.

💻 기초 코드 예시 (충돌 감지)
C#
using UnityEngine;

public class CollisionDetector : MonoBehaviour
{
    // [유형 1] 진짜로 부딪혔을 때 (벽에 부딪힘, 바닥에 착지함 등)
    // collision 파라미터: 나와 부딪힌 상대방의 정보가 들어옵니다.
    private void OnCollisionEnter(Collision collision)
    {
        // 만약 부딪힌 상대방의 태그가 "Wall" 이라면?
        if (collision.gameObject.CompareTag("Wall"))
        {
            Debug.Log("벽에 쿵 부딪혔다!");
        }
    }

    // [유형 2] 겹치거나 통과했을 때 (아이템 먹기, 독 구역 진입 등)
    // other 파라미터: 나를 통과해간 상대방 콜라이더 정보가 들어옵니다.
    private void OnTriggerEnter(Collider other)
    {
        // 만약 통과한 오브젝트의 태그가 "Item" 이라면?
        if (other.CompareTag("Item"))
        {
            Debug.Log("아이템 획득!");
            Destroy(other.gameObject); // 아이템 삭제
        }
    }
}
⚠️ 중요 조건: 충돌 이벤트(OnCollision, OnTrigger)가 발생하려면, 두 오브젝트 모두 콜라이더가 있어야 하고, **최소한 어느 한쪽에는 반드시 리지드바디(Rigidbody)**가 붙어 있어야 작동합니다!

3. 레이캐스트 (Raycast)
특정 위치에서 원하는 방향으로 가상의 선(Ray)을 쏘아, 그 선에 부딪히는 오브젝트가 있는지 알아내는 기술입니다.

📌 어디에 배치하나요?
선(레이저)을 쏘아서 판단을 내려야 하는 주체(총을 쏘는 플레이어, 바닥 검사를 하는 캐릭터, 전방의 장애물을 감지하는 적 등)의 스크립트에 들어갑니다.

💻 기초 코드 예시 (전방 장애물 및 바닥 감지)
C#
using UnityEngine;

public class RaycastLaser : MonoBehaviour
{
    public float maxDistance = 10f; // 레이저를 얼마나 멀리 쏠 것인가

    void Update()
    {
        // 1. 부딪힌 대상의 상세 정보를 담을 빈 상자(RaycastHit)를 만듭니다.
        RaycastHit hit;

        // 2. 어디서(내 위치), 어느 방향으로(내 앞쪽) 레이저를 쏠지 정의합니다.
        Vector3 origin = transform.position;
        Vector3 direction = transform.forward;

        // 에디터 뷰에서 개발자가 눈으로 볼 수 있게 빨간 선을 그려줍니다. (실제 게임엔 안 보임)
        Debug.DrawRay(origin, direction * maxDistance, Color.red);

        // 3. 레이저를 발사합니다! (부딪히면 true, 안 부딪히면 false 반환)
        if (Physics.Raycast(origin, direction, out hit, maxDistance))
        {
            // hit 상자 안에 부딪힌 녀석의 정보가 들어있습니다.
            Debug.Log("레이저에 맞은 오브젝트 이름: " + hit.collider.gameObject.name);
            Debug.Log("부딪힌 정확한 좌표: " + hit.point);
            
            // 만약 맞은 녀석이 적이라면?
            if (hit.collider.CompareTag("Enemy"))
            {
                Debug.Log("적을 조준했다!");
            }
        }
    }
}
3줄 요약 핵심 정리
Rigidbody (rb.AddForce, rb.velocity)

오브젝트를 물리적으로 움직이게 만들 때 씁니다.

Collider (OnCollisionEnter, OnTriggerEnter)

오브젝트들이 서로 부딪히거나 스쳐 지나갈 때 이벤트를 처리합니다.

Raycast (Physics.Raycast(위치, 방향, out hit, 거리))

보이지 않는 선을 레이저처럼 쏴서 특정 거리 안에 무엇이 있는지 조준/감지할 때 씁니다.
